# Kapitel 6 – Quadratische Gleichungen: Lehrer-Stil Beispiele (für Pattern-Lernen)
Hinweis: Das sind **Muster**, keine offizielle Lösungssammlung. Ziel: Format und “Schritt-Logik” trainieren.

============================================================
BEISPIEL 1 — pq-Formel (a=1)
============================================================

D: R

Schritt 1
x^2 - 6x + 5 = 0
p = -6
q = 5

Schritt 2
x = (-p 2tel) + √( (p 2tel)^2 - q )   [pq]
x = (-p 2tel) - √( (p 2tel)^2 - q )   [pq]

Schritt 3
x = 3 + √(9 - 5)   [rechnen]
x = 3 - √(9 - 5)   [rechnen]

Schritt 4
x = 3 + 2
x = 3 - 2

Schritt 5
x = 5
x = 1

L: {x1 = 5, x2 = 1}

============================================================
BEISPIEL 3 — Mitternachtsformel (a≠1)
============================================================

D: R

Schritt 1
2x^2 - 4x - 6 = 0
a = 2
b = -4
c = -6

Schritt 2
Δ = b^2 - 4·a·c   [Δ]
Δ = 16 - 4·2·(-6) [rechnen]
Δ = 16 + 48
Δ = 64

Schritt 3
x = (-b + √Δ) (2a)tel   [Mitternacht]
x = (-b - √Δ) (2a)tel   [Mitternacht]

Schritt 4
x = (4 + 8) 4tel
x = (4 - 8) 4tel

Schritt 5
x = 12 4tel
x = -4 4tel

Schritt 6
x = 3
x = -1

L: {x1 = 3, x2 = -1}

============================================================
BEISPIEL 4 — Quadratische Ergänzung → dann √
============================================================

D: R

Schritt 1
x^2 + 4x - 1 = 0
x^2 + 4x = 1   [+1]

Schritt 2
x^2 + 4x + 4 = 1 + 4   [+4]

Schritt 3
(x + 2)^2 = 5   [binom]

Schritt 4
x + 2 = √5    [√]
x + 2 = -√5   [√]

Schritt 5
x = √5 - 2    [-2]
x = -√5 - 2   [-2]

L: {x1 = √5 - 2, x2 = -√5 - 2}

============================================================
BEISPIEL 5 — Nenner → Hauptnenner → quadratisch
============================================================

D: R \ {2}

Schritt 1
1 (x-2)tel = x

Schritt 2
1 = x(x-2)   [·HN]

Schritt 3
1 = x^2 - 2x
0 = x^2 - 2x - 1   [-1]

Schritt 4
p = -2
q = -1   [a=1]

Schritt 5
x = (-p 2tel) + √( (p 2tel)^2 - q )   [pq]
x = (-p 2tel) - √( (p 2tel)^2 - q )   [pq]

Schritt 6
x = 1 + √(1 + 1)
x = 1 - √(1 + 1)

Schritt 7
x = 1 + √2
x = 1 - √2

Schritt 8
(D check) x ≠ 2   [ok]

L: {x1 = 1 + √2, x2 = 1 - √2}
