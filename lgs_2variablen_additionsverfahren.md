## 1. Wann Additionsverfahren?

- Wenn sich eine Variable durch Addition oder Subtraktion der Gleichungen leicht eliminieren lässt.
- Wenn nach geeigneter Multiplikation die Koeffizienten einer Variable gleich oder gegensinnig sind.
- Standardverfahren, wenn Gleichungen symmetrisch wirken oder nach Entfernen von Nennern gute Koeffizienten entstehen.

---

## 2. Standardschema

Gegeben:

```text
(1) ...
(2) ...
Vorgehen:

text
Code kopieren
D: ...

Schritt 1
Gleichungssystem aufschreiben:
(1) ...
(2) ...

Schritt 2
Falls nötig, eine oder beide Gleichungen mit Faktoren multiplizieren, um gleiche/entgegengesetzte Koeffizienten zu erhalten:
...   [·k]

Schritt 3
Gleichungen addieren oder subtrahieren, sodass eine Variable wegfällt:
...   [+] oder [−]

Schritt 4
Die erhaltene Gleichung nach der verbleibenden Variablen lösen:
...   [Umformungen]

Schritt 5
Die gefundene Variable in eine der ursprünglichen Gleichungen einsetzen:
...   [Einsetzen]

Schritt 6
Lösungsmenge angeben:
L: {(x|y) = (...|...)}
3. Beispiel
Gegeben:

text
Code kopieren
(1) 2x + y = 7
(2) x − y = 1
Lösung mit Additionsverfahren:

text
Code kopieren
D: ℝ

Schritt 1
(1) 2x + y = 7
(2) x − y = 1

Schritt 2
(2) mit 1 multiplizieren (keine Änderung nötig), direkt addieren möglich.

Schritt 3
(1) + (2):
(2x + y) + (x − y) = 7 + 1       [+]
3x = 8                           [:3]
x = 8 3tel

Schritt 4
x in (2) einsetzen:
x − y = 1
8 3tel − y = 1                   [−8 3tel]
−y = 1 − 8 3tel
−y = (3 3tel − 8 3tel)
−y = −5 3tel                     [:−1]
y = 5 3tel

Schritt 5
L: {(x|y) = (8 3tel|5 3tel)}
4. Typische Fehlerquellen
falscher Multiplikationsfaktor

falsche Vorzeichen beim Addieren/Subtrahieren

in Schritt 3 falsche Gleichungen kombiniert

L nicht in der richtigen Form angegeben
